# Digital Landman Network

A single-page website for digitallandman.io - Texas Rural Real Estate Marketplace.

## Local Development

Simply open `index.html` in your browser to view the site locally.

## Deployment

### Via Vercel CLI
```bash
vercel deploy
```

### Via GitHub Desktop
1. Clone this repository
2. Make changes
3. Commit and push via GitHub Desktop
4. Connect to Vercel/Netlify for automatic deployments

## Tech Stack
- HTML5
- Tailwind CSS (via CDN)
- No build process required
